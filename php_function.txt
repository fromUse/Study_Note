# is_int() ,is_integer() 判断某个变量是否是数字 或者 数字字符串 [ 1 /true, false ] 数字字符串不起用

# is_flaot(),is_double() 判断某个变量是否是浮点数 [ 1/true,false ] 数字字符串不起用
